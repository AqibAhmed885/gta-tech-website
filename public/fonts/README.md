# Brandon Grotesque Font Files

Place your Brandon Grotesque font files in this directory with the following names:

- `BrandonGrotesque-Light.woff2` (weight 300)
- `BrandonGrotesque-Regular.woff2` (weight 400)
- `BrandonGrotesque-Medium.woff2` (weight 500)
- `BrandonGrotesque-Bold.woff2` (weight 700)

## Where to get Brandon Grotesque

Brandon Grotesque is a premium font by HVD Fonts. You can purchase it from:
- https://www.myfonts.com/collections/brandon-grotesque-font-hvd-fonts
- https://fonts.adobe.com/fonts/brandon-grotesque (if you have Adobe Fonts subscription)

## Converting to WOFF2

If you have OTF or TTF files, you can convert them to WOFF2 format using:
- https://cloudconvert.com/woff2-converter
- https://transfonter.org/

WOFF2 provides the best compression and is supported by all modern browsers.

## Alternative: Use a similar free font

If you don't have access to Brandon Grotesque, you can use these similar free alternatives:
- **Montserrat** (Google Fonts) - geometric sans-serif
- **Nunito Sans** (Google Fonts) - rounded geometric sans-serif
- **DM Sans** (Google Fonts) - clean geometric sans-serif

To use a Google Font instead, update `app/layout.tsx` to import from `next/font/google`.
